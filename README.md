# 王者：弱水三千

## 简介

**类型**：王者荣耀同人 + 洪荒穿越 + 群像史诗

**主角**：方长，28 岁企业战略扩张部负责人，穿越后 18 岁，无金手指，智力型主角

**核心设定**：
- 洪荒末期仙神跨界来到苍澜大陆，建立倒悬天统治
- 人族 10%、半魔<1%、魔种 90%，三族分立
- 武道七境：后天→先天→登神→斩大罗→不望天门→弱水三千→大道同归
- 六女主平等，无正宫侧妃之分

**篇幅规划**：280-350 万字 / 700-850 章（实际大纲 996 章）

**更新频率**：日更

## 世界观速查

**世界名**：苍澜大陆

**种族**：
- 人族（10%）：被倒悬天统治，抱团生存
- 半魔（<1%）：人魔混血，被歧视打杀
- 魔种（90%）：本土生命，占据大部分地域
- 仙神（<1%）：顶层统治者

**力量体系**：
- 洪荒：凡人→地仙→天仙→真仙→玄仙→金仙→太乙金仙→大罗金仙→准圣→圣人
- 武道：后天→先天→登神→斩大罗→不望天门→弱水三千→大道同归
- 魔种：1-10 阶（最高 9 阶，10 阶已灭绝）
